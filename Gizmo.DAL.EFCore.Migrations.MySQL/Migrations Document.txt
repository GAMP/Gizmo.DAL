- To add new migration:

	1- Configure the dbcontext as the following :

	   connectionString = "Server={ServerName};Database={DatabaseName};userId={UserId};password={Password}";

       builder.Services.AddDbContext<ApplicationDbContext>(options => 
	   options.UseMySql(connectionString, ServerVersion.AutoDetect(connectionString), x => x.MigrationsAssembly("Gizmo.DAL.EFCore.Migrations.MySQL"));
	   
	2- Open package manager console.
	3- Select the default project Gizmo.DAL.EFCore.Migrations.MySQL
	4- Enter the command line > EntityFrameworkCore\Add-Migration {MigrationName}
